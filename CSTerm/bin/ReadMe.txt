프로그램 실행방법

1. Project.exe 파일을 실행한다

2. 이름을 입력한다.(8자 이상은 짤림)

3. 점수를 입력한다.(100 초과는 다시입력)

4. 학생입력이 끝났으면 이름 공백, 점수 -1을 입력한다.
   (최대 100명까지 입력가능)

5. 정렬된 값과 표준편차 값을 확인한다.
